Linda's World Quiz

Some geographic mini-games

International

Find The Country

Find the country with the name displayed in the upper right corner. Click or tap on the country shape. The country is highlighted green if you clicked the correct country. You can play the game in multiplayer mode. If each player enters the same text before starting the game, they all get the same countries. The player which finds the country first wins the round.

Find The City

Find the city with the name displayed in the upper right corner. Click or tap on the icon. The icon is highlighted green if you clicked the correct city. You can play the game in multiplayer mode. If each player enters the same text before starting the game, they all get the same cities. The player which finds the city first wins the round.

Locate The City

Try to locate the the city. Place the marker on the map. After you click "Solve", the distance to the real location of the city is displayed. You can play the game in multiplayer mode. If each player enters the same text before starting the game, they all get the same cities. You can compare your distances then.

Deutsch

Finde die Stadt

Finde die Stadt die oben rechts angezeigt wird. Ihr könnt das Spiel auch gegeneinander mit mehreren Computern oder Smartphones spielen. Wenn jeder Spieler denselben Text unten eingibt, bekommen alle die gleichen Städte angezeigt. Wer die Stadt zuerst findet gewinnt die Runde.

Orte die Stadt

Versuche de Lage der Stadt in der Karte zu markieren. Wenn du auflösen drückst wird der Abstand zur richtgen Position angezeigt. Ihr könnt das Spiel auch gegeneinander mit mehreren Computern oder Smartphones spielen. Wenn jeder Spieler denselben Text unten eingibt, bekommen alle die gleichen Städte angezeigt. Dann könnt ihr die Entfernungen vergleichen.
